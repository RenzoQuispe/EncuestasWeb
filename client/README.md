## Proceso de construccion del frontend

### Inicializar Angular
```bash
npx @angular/cli new client --routing --style=css
cd client
npm install
npx ng serve
```
## Comandos

```bash
ng generate component component-name
```
```bash
ng generate --help
```
```bash
ng build
```
```bash
ng test
```
